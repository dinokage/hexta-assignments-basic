Project Overview:

This is a dynamic and responsive news web application built using HTML, CSS, JavaScript, and React.js. The app aggregates news from various sources through a public API and displays it in an easy-to-read, user-friendly layout. The focus of the project is to provide users with quick access to the latest news articles, while ensuring that the design is clean, responsive, and interactive.

  • Error Handling: Implements graceful error handling for API fetches, providing users with a friendly message if the data fails to load
