# 🧑‍💻 GitHub Submission Instructions

Follow these steps to **fork the repository**, **clone the repository**, **create your own branch**, **make changes**, and **push it to GitHub**.

---

## ✅ Prerequisites

Make sure you have:
- A **GitHub account**
- Git installed on your system
- Access to the repository link (e.g., provided by the instructor)

---

## 📥 1. Fork and Clone the Repository

Click on the fork button in top right corner

Open a terminal and run:

```bash
git clone https://github.com/<your-username>/hexta-assignments-basic.git
```

Navigate into the project folder:

```bash
cd hexta-assignments-basic
```

---

## 🌿 2. Create a New Branch with Your Name

Run the following command, replacing `your-name` with your actual name (use dashes instead of spaces):

```bash
git checkout -b your-name
```

**Examples:**

```bash
git checkout -b john-doe
```

---

## ✏️ 3. Make Your Changes

- Open the project in your code editor (VS Code, etc.)
- Add or modify the files as per the instructions
- Save your changes

---

## ✅ 4. Stage and Commit Your Changes

Run the following commands:

```bash
git add .
git commit -m "Added my changes - John Doe"
```

---

## 🚀 5. Push Your Branch to GitHub

```bash
git push origin your-name
```

**Example:**

```bash
git push origin john-doe
```

---

## 📬 6. Create a Pull Request (PR)

1. Go to the repository on GitHub
2. You’ll see a **“Compare & pull request”** button – click it
3. Add a description if needed
4. Click **Create pull request**

---

## 🎉 You’re Done!

Once your PR is reviewed and merged, your submission will be complete.  
If you face any issues, reach out to the instructor or TA.
